### 1.本项目（青出于蓝家教网）使用教程
####  （1）下载本项目；
####  （2）剪切（移除）项目中的.sql文件（数据库及表创建文件），并执行该文件；
####  （3）剪切（移除）项目中的familyeduFile（项目图片文件）目录至项目的同级目录；
####  （4）在properties文件配置mysql数据库；
####  （5）运行项目，并在浏览器地址栏数据localhost回车（登录用户可在数据中查询到，admin为管理员）。

### 2.项目功能简介
####  （1）招聘信息、简历信息发布与管理；
####  （2）个人信息管理；
####  （3）在线聊天；
####  （4）用户管理；
####  （5）主页轮播图片管理等等。
